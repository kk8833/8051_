# 8051 Embedded C Bare-Metal Programming

This repository contains my learning and implementation of 8051 Microcontroller Programming using Embedded C with a Bare-Metal Approach. The objective is to understand the internal architecture of the 8051 and develop firmware by directly accessing hardware registers without relying on external libraries or frameworks.

## About

The 8051 microcontroller is one of the most widely used embedded systems platforms and serves as a strong foundation for learning microcontroller architecture, peripheral interfacing, and low-level programming concepts.

In this repository, all programs are written in Embedded C and focus on direct register manipulation, providing a deeper understanding of how microcontrollers operate at the hardware level.

## Topics Covered

* GPIO Programming
* LED Blinking
* Switch Interfacing
* Delay Generation
* Timers and Counters
* Interrupts
* UART Serial Communication
* LCD Interfacing
* Seven Segment Display
* Keypad Interfacing
* ADC Interfacing
* Embedded C Fundamentals


## Learning Objectives

* Understand the architecture of the 8051 microcontroller.
* Develop firmware using Embedded C.
* Learn direct register-level programming.
* Understand peripheral interfacing techniques.
* Build a strong foundation for embedded systems development.
* Improve debugging and problem-solving skills in embedded applications.

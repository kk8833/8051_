# LED Blinking Using 8051 Microcontroller

## Overview

This project demonstrates the basic GPIO operation of the 8051 microcontroller by blinking two LEDs connected to Port 1 pins P1.0 and P1.1. The program is written in Embedded C using a bare-metal approach, directly controlling the port pins through register manipulation.

This is one of the fundamental embedded systems projects and serves as an introduction to microcontroller programming, digital output control, and delay generation.

## Objective

* Learn basic GPIO programming on the 8051.
* Understand how to configure and control Port 1 pins.
* Generate software delays for visible LED blinking.
* Verify the functionality using Proteus simulation.

## Simulation

The circuit was designed and tested using Proteus. Both LEDs connected to P1.0 and P1.1 successfully blinked according to the programmed delay, confirming correct GPIO operation.
